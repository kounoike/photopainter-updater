# サーバー設定コントラクト: ディザリングアルゴリズム切り替え

**Branch**: `017-dithering-quality` | **Date**: 2026-03-30

## 環境変数インターフェース（評価期間中のみ）

### `DITHER_USE_ATKINSON`

| 項目 | 内容 |
|------|------|
| 型 | 文字列（`"1"` で有効、未設定または他の値で無効） |
| デフォルト | 無効（Floyd-Steinberg を使用） |
| 効果 | Atkinson 誤差拡散アルゴリズムを使用する |
| クリーンアップ後 | 削除 |

### `DITHER_USE_LAB`

| 項目 | 内容 |
|------|------|
| 型 | 文字列（`"1"` で有効、未設定または他の値で無効） |
| デフォルト | 無効（RGB ユークリッド距離を使用） |
| 効果 | 色距離計算を CIE Lab 色空間（CIE76）で行う |
| クリーンアップ後 | 削除 |

## 既存環境変数（変更なし）

| 変数 | 内容 |
|------|------|
| `PORT` | サーバーポート番号（デフォルト: 8000） |
| `CONTENT_DIR` | 画像ソースディレクトリ |

## HTTP エンドポイント（変更なし）

ディザリングアルゴリズムの変更は HTTP インターフェースに影響しない。全エンドポイントのレスポンス形式・ステータスコードは変更前と同一。

| エンドポイント | Content-Type | 変更 |
|--------------|-------------|------|
| `GET /` | `application/vnd.photopainter-frame` | なし |
| `GET /image.bin` | `application/vnd.photopainter-frame` | なし |
| `GET /image.bmp` | `image/bmp` | なし |
